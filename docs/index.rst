Welcome to UC Merced's ACM Chapter Website. This documentation
serves as a guide for the website's development and maintenance.

Purpose
=======

The UCMACM Website serves as a central hub for all things ACM at UC Merced.
It contains information about the club, its members, and its events. On top of
those, it acts as a dashboard for members to book, and register for workshops and events.
